LoTuS@Runtime
================================

1. About the LoTuS@Runtime
--------------------------------------
The Lotus@Runtime is a tool to perform, in an integrated manner, monitoring, verification, and violations notification found during execution of a self adaptive system.

2. Dependencies
--------------------------------------

- Maven 3.x
